Declare for a Form :
  <!--html declaration-->
  • <form name="reviewForm" ng-controller="ReviewControler as reviewCtrl" ng-submit="reviewCtrl.addReview(product)" >
  • //JavaScript declaration
    app.controller('ReviewController', function(){
     this.review = {};
     this.addReview = function(product){
      product.review.push(this.review); //To add the insert review in the reviews tab
      this.review = {}; //To reset the form element
     }
    });
Form validation :
  <!--html declaration-->
  • novalidate to not use the html validation
  • add in the ng-submit tag : reviewForm.$valid && to control if the form is valid to allow the submission
  • use min and max to define set the valid size of a field:
    min="1" max="5"
  • ng-include="'product-title.html'"
  • Add a css format with the validation :
     .ng-invalid.ng-dirty{
      border-color: red;
     }

     .ng-valid.ng-dirty{
      border-color: green;
     }
Include a page :
  • for a basic include add this in an html tag where you want include the page: ng-include="'product-title.html'"
  • Element declaration : <product-title></product-title>
  • Attribute declaration : <h3 product-title></h3>
  • //JavaScript declaration
    app.directive('productTitle', function(){
     return {
      //this
      restrict: 'E', //Stand for Element
      //or that
      restrict: 'A', //Stand for Attribut
      templateUrl: 'product-title.html'
     };
    });
  • In the included page product-title.html :
    <!--html declaration-->
    <h3>
     {{product.name}}
     <em class="pull-right">{{product.price | currency}}</em>
    </h3>
  • Add a controller to the directive :
    controller:function(){//write the behavior of the controller here}, controllerAs: 'panels'//write the controller name to use in the page
